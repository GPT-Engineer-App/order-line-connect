# order-line-connect

A collaborative webapp for staff of a restobar for the orders due to the problem that the kitchen is detached from the restobar. An internal ordering system as a line of communication for the status of the order input

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/order-line-connect.git
cd order-line-connect
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
